模仿抽屉    
==
作者：王宇夫     

## 程序介绍:     
    使用python3+、django1.10+，Windows、Linux环境使用；    
	实现了抽屉的基础功能。     
	
## 程序使用：    
	
	1、导入sql    
	2、运行django     
	3、访问/     
	4、实现内容：     
		登录：root/123    
		发布new（ajax）    
		点赞（ajax）     
		如果未登录，点赞和评论时会提示登录。    
		新加：    
			验证码、多级评论、优化点赞效果
